When the robot reaches a waypoint, the node calls the \textquotesingle{}request\+\_\+set\+\_\+orientation\textquotesingle{} service to adjust the robotÂ’s orientation for optimal arm movement. Once oriented, the robot is instructed to move its arm using the \textquotesingle{}request\+\_\+move\+\_\+arm\textquotesingle{} service and collect hints. If hints are successfully collected, the service returns \textquotesingle{}true\textquotesingle{}, and the robot proceeds to the next waypoint.
